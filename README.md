# Technologies Used
- **React**
- **Tailwindcss**
- **json-server**


# How to Run

clone the repository
HTTPS
```bash
  git clone https://github.com/Devalyo/aws_react_dez24_d03
```


install the dependencies
```bash
npm install
```

then start the application with: 

```bash
npm run dev
```
Done, the application and database will start running on localhost.
